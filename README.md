# Achei Quem Queira (AQQ)

Uma plataforma de desapego local voltada para economia circular e reaproveitamento comunitário.

## Objetivo

Facilitar a circulação de itens entre moradores próximos, incentivando reutilização, sustentabilidade urbana e redução do descarte desnecessário.

## Características

- Publicação rápida de itens
- Sistema de qualidade padronizado
- Retirada local
- Negociação simplificada
- Interface mobile-first
- Economia circular

## Stack

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Vanilla JavaScript
- LocalStorage

## Arquitetura

O projeto utiliza uma arquitetura modular baseada em:

- Core
- Componentes Base
- Componentes do Produto
- Páginas
